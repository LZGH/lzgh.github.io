emacs基本快捷键

发表于 2016-10-31   |   分类于 emacs   |     |   阅读次数

为了方便安利emacs，列一些最基本快捷键。

约定

C 表示Ctrl，M 表示Alt，S 表示Shift
C-x C-f 表示先按下Ctrl键和x，再按下Ctrl键和f键
M-S-RET 表示同时按下Alt、Shift和回车键

基本命令

emacs最小化：C-z
退出emacs：C-x C-c
取消未完成的命令：C-g

文件操作

打开文件：C-x C-f
保存文件：C-x C-s
创建文件：C-x C-f 输入文件名回车确认
另存为：C-x C-w

编辑快捷键

复制：M-w
粘贴：C-y
剪切：C-w
全选：C-x h

运行代码快捷键

运行：C-c
编译(java)：C-c

缓冲区操作

前一个buffer：C-x ←
后一个buffer：C-x →
buffer列表：C-x b

窗口操作

保留一个窗口：C-x 1
垂直分割窗口：C-x 2
水平分割窗口：C-x 3

org模式快捷键

循环切换光标所在大纲状态：TAB
循环切换整个文档大纲状态：S-TAB
插入同级列表项：M-RET
插入同级TODO列表项：M-S-RET
改变列表项层级关系：M-left/right
上下移动列表项：M-up/down
TODO/DOWN切换：S-left/right
改变任务等级：S-up/down
